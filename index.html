HTML, which stands for HyperText Markup Language, is a markup language that developers use to structure contents on their webpage. It contains tags that are used to define specific things. It is easy and can be made fun to learn.
On the other hand, CSS, which stands for Cascading Style Sheets, is another tool used by the developers to style those same contents on the webpage. These two tools are used hand in hand with Javascript to create a very unique and wonderful website.
On this page, you'll get to learn some of these basic tags used in almost all webpages.

BASIC HTML TAGS!!!

Here is a list of some of the HTML tags we know:

  • The <!DOCTYPE> tag defines the document type.
  • The <html> and </html> tags which is the root of all html documents.
  • The <head> and </head> tags which contains information about the document.
  • The <title> and </title> tags
    • which defines the title of the site.
  • The <p> and </p> tags representing paragraphs.
  • The <br /> tag used for line breaks.
  • The <ul> and </ul> tags representing unordered lists.
  • The <ol> and </ol> tags representing ordered lists.
  • The <img> tag used to insert images.
  • The <body> and </body> tags which contain the body of the document.
  • The <table> and </table> tags representing tables.
  • The <div> and </div> tags representing a division of the document.
  • The <a> and </a> tags for creating hyperlinks.
  • The <b> and </b> tags for making words bold.
    • Therefore, using some of these tags, here is an example of a html document

    <html>
    <head>
    <title>     First page </title>
    </head>

  • The <h1>-<h6> and </h1>-</h6> tags representing the heading levels with h1 being the biggest and h6 the smallest.
  • The <header> and </haeder> tags which define header for a document.
  • The <footer> and </footer> tags which define a footer for a document.
  • The <li> and </li> tags for list items.
  • The <strong> and </strong> tags which define an important text.
  • The <em> and </em> tags which define an emphasised text.
  • The <pre> and </pre> tags which define preformatted text.
  • The <button> and </button> tags which define a clickable button.
  • The <aside> and </aside> tags which define a content aside from the main content.
  • The <nav> and </nav> tags which define navigation links.
  • The <section> and </section> tags define a section in a document.
  • The <small> and </small> tags define smaller text.

These are just a few tags used in a html document.

CSS- Cascading Style Sheets


Here is a screenshot of an example of a CSS code.

CSS is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the world wide web, like HTML and JavaScript.

CSS Syntax

A CSS rule-set consists of a selector and declaration block, which includes a property and a value. Here is an example:

h1
{color:blue;}


In the example given above, the h1 is called the selector and the elements in the curly brackets are the Declaration , where color is the Property and blue is the Value.

More about coding.

Let's tell you a little more about coding as much as we know it. Coding is the process of using a programming language to get a computer to behave how you want it to. Every line of code tells the computer what to do and a document filled with lines of code is called a script. Every script is designed to carry out a job.

  1. Definition of coding and programming.
  2. Give a clear difference about the above mentioned and also a little more knowledge as we journey together.

Though the terms are often used interchangeably, coding and programming are two different things. Coding refers to writing codes for application, while programming is a much broader term.
Programmers find solutions to problems and determine how they are to be solved.
Programmers generally deal with the big picture within an application, not just categorized lines of code.

Computer programming is the process of designing and building an executable computer program to accomplish a specific computing result or to perform a specific task .
The purpose of programming is to find a sequence of instructions that will automate the performance of a task [which can be as complex as an operating system] on a computer, often for solving a given problem.

To simply put it, coding is interacting with the computer.
People use coding to give computers and other machines instructions on what actions to perform. Furthermore, we use coding to program the websites, apps and other technologies we interact with everyday.
It helps you build applications, put them over the servers and use so that muliple users can use and get the benefits.

There are 3 types of codes, which are namely,

  • Feature code,
  • Infrastructure code and
  • Reliability code.

  • Feature code:
    • It is simply a code that can perform specific action. This could rage from pausing your phone in a queue or having someone's number on speed dial.

  • Infrasturcture code:
    • This is the process of managing and provisioning computer data centers through machine-readable definition files rather than physical hardware configurations or interactive configuration tools.

  • Reliability code:
    • This is an attribute of any computer related component that consistently performs according to its specifications. It has long been considered as one of three related attributes that must be considered when making, buying or using a computer product or component.
